# ホットクック献立アプリ

## 目的
忙しい子育て世帯向けに、
ホットクックで作れる夕食を1週間分提案する。

## 対象
4人家族

## 機能

### 献立作成
- 1週間分の夕食献立を作成
- 月火木金は冷凍ミールキット方式
- 水土日は通常調理

### レシピ表示
- 材料
- 分量
- 調理時間
- ホットクック設定
- 保存期間

### 買い物リスト
- 1週間分を自動集計
- 肉、魚、野菜、調味料に分類

### スマホ対応
- iPhone中心
- PWA対応
- ホーム画面追加可能

## MVP（最初に作るもの）
- 固定レシピ20品
- 1週間献立表示
- 買い物リスト表示
